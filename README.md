# Installation

Things you need:

- A Computer
- Node.js
- git console git-scm.com
- npm

### in the git console type:

> $ git clone https://github.com/77ZTM/electron-example-app.git

> $ cd electron-example-app

> $ npm start

Those commands should open a window
